This project explores job market data, extracts actionable insights, and visualises the findings using interactive dashboards. The goal is to equip stakeholders (job seekers, recruiters, educational institutions) with data-driven intelligence on hiring trends, skill demands, and job availability.
Aimed at educating and preparing people like myself looking to kick start their data career on what to expect for the various jobs that involve data.

## 1. Data Engineering

Acquired a data on data jobs across the world.
The dataset has the different data jobs such as Data Scientist, Data Analyst, Data Engineers etc.
Clean the data filling any missing values, removing duplicates and anomalies

## 2. Exploratory Data Analysis (EDA)
Includes:

Identifing skills required by the different data jobs
Categorizing the various data jobs and their locations
Analyzing the salary ranges for each data job
Grouping the data jobs by location.

## 3. Visualizations & Dashboards

Matplotlib visuals on the various trends. 
Tableau dashboard that contains:
  1. Job distrubution on the world map.
  2. KPIs on salary average and rates per hour
  3. Companies hiring
  4. New Dashboard that gets into specifics of each data job with feature like work mode (remote vs onsite)
